Tier 1 teaches the stack with no autonomy; Tier 2 introduces the agentic loop & tools; Tier 3 is multi-agent & long-horizon.

Tier 1 · Foundations

start here · no autonomy

Deterministic pipelines — one/few LLM calls, no agent loop. You learn RAG, structured outputs, evals, and cost without the complexity of autonomy.

P1 · 📄 Docs Q&A Assistant (RAG) ★ recommended first

Answer questions over a company's own docs/knowledge base, grounded & cited — the "hello world" of production AI, and the base of half the apps out there.
RAGembeddingsvector storechunkingcitationsgrounding
Teaches: the whole retrieval stack · Maps to D5, D3
⚠ Prod failures: stale index · bad chunking · low retrieval recall · confident hallucination · cost of re-embedding

P2 · 📝 Meeting Notes → Action Items

Turn a transcript into a structured summary + owned action items with due dates — a pure workflow, no agent needed.
structured outputsworkflow vs agentbatch APIevals
Teaches: when NOT to use an agent · Maps to D3, D1, D5
⚠ Prod failures: schema drift · long transcripts overflow context · silent extraction misses

P3 · 🏷️ Content Classifier at Scale

Classify/moderate a high volume of items (tickets, posts, reviews) cheaply and reliably.
model-tier choicebatchrate limitsevalsdrift
Teaches: cost/throughput engineering · Maps to D5
⚠ Prod failures: throughput vs rate limits · label drift · prompt-injection in user content · eval rot

Tier 2 · Agentic core

the agent loop + tools

Now the model decides and acts: tool use, structured actions, error-recovery loops, and human-in-the-loop for anything risky.

P4 · 🎧 Customer Support Agent ★ recommended 2nd

Resolve support requests: look up an order, check status, issue a refund — with human approval on the irreversible bits.
agentic looptool designhuman-in-the-loopreversibilityguardrailsescalation
Teaches: the core agent + safe actions · Maps to D1, D4
⚠ Prod failures: wrong refund issued · tool 400s break the loop · prompt injection triggers an action · runaway retries

P5 · 🗄️ NL→SQL Analytics Copilot

Turn "how many EU orders last quarter?" into SQL over a warehouse, run it, and answer — read-only and safe.

Extract structured fields from invoices/forms (PDF/images) into a validated schema, at scale.

1. 📄 P1 · Docs Q&A (RAG) — learn the retrieval stack & grounding (no autonomy).

3. 🔬 P7 · Research Agent — multi-agent orchestration, parallelism, verification.
Then add 📊 P10 · Eval Harness as a companion — because in production, "how do you know it works?" is the question that never goes away.
